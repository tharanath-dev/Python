# 🎓 Student Record System (Python)

A simple and user-friendly command-line based **Student Record Management System** built using Python.  
This micro-project helps manage student data such as personal details and academic performance efficiently.

---

## 📌 Project Information
- 🎯 **Project Type**: Micro-Project (First Year Python)

## 🚀 Features

- ➕ Add new student records  
- 🔍 Search students by Roll Number or Name  
- ✏️ Update student details (Name, Phone, Department, CGPA)  
- 📋 Display all student records  
- ❌ Delete student records  
- ✅ Input validation for CGPA (0.0 – 10.0)  

---

## 🛠️ Technologies Used

- Python (Core Programming)
- Data Structures (Lists & Dictionaries)
- Loops & Conditional Statements
- Functions (Modular Programming)
